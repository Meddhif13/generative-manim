# Streamlit Interface

This directory contains a lightweight Streamlit interface for interacting with the Generative Manim API.

## Installation

```bash
pip install -r interface/requirements.txt
```

## Usage

Launch the app with:

```bash
streamlit run interface/app.py
```

Enter one prompt per line, choose a narrative style and model, then generate code, chat to improve it, render videos and export the resulting dataset as JSONL.
